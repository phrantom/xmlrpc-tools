# xmlrpc-tools
Bash scripts for XMLRPC API Testing

* xmlrpc_test.sh
  
```bash
./xmlrpc_test.sh
Usage: ./xmlrpc_test.sh [ option ]
	-s [ Url ] 
	-l [ Urls File ]
```

* xmlrpc_pingback.sh

```bash
./xmlrpc_pingback.sh
Use: ./xmlrpc_pingback.sh [ url-vulnerable ] [ url-to-ping ]
```


* xmlrpc_extract_methods.sh

```bash
./xmlrpc_extract_methods.sh
 Usage ./xmlrpc_extract_methods.sh -u [ url-vulnerable ]
```
